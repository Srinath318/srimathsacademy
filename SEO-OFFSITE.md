# SEO checklist (outside the website)

Do these steps after deploying the site to **https://srimathsacademy.in**

## 1. Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console).
2. Add property: `https://srimathsacademy.in`
3. Verify ownership (HTML file upload or DNS — GitHub Pages users often use DNS TXT).
4. Submit sitemap: `https://srimathsacademy.in/sitemap.xml`
5. Check **Pages** and **Indexing** weekly; fix any errors.

## 2. Google Business Profile

1. Create or claim a profile at [Google Business](https://business.google.com).
2. Category: **Tutoring service** or **Education center**.
3. Add phone: **+91 94869 81357**, website, and photos.
4. In description, say **online 1-on-1 maths tuition**, Classes 6–12, CBSE/ICSE/State.
5. Set **service area**: India (add cities you actively serve, e.g. Chennai, Coimbatore).
6. Post updates weekly (demo offer, exam tips).

## 3. Reviews

- After a successful demo or month of classes, ask parents for a **Google review**.
- Reply to every review politely.

## 4. Backlinks (links from other sites to yours)

- Tutor LinkedIn profile → link to srimathsacademy.in
- WhatsApp status / school parent groups (when allowed)
- Local education directories (quality sites only)
- Avoid buying cheap spam links

## 5. Content marketing

- Short YouTube or Instagram Reels: one maths tip per week
- Link to your site and demo booking in every video description
- Tamil captions for Tamil Nadu reach

## 6. Analytics

- Add [Google Analytics](https://analytics.google.com) or [Plausible](https://plausible.io) to see which pages get traffic
- Use Search Console **Performance** report to see real search queries

## 7. Realistic timeline

| Goal | Typical time |
|------|----------------|
| Site indexed | 1–4 weeks |
| Rank for "Sri Maths Academy" | Weeks |
| Rank for long-tail ("online CBSE class 10 maths tutor") | 3–12+ months |
| Rank #1 for "maths online tuition" | Very hard; needs strong backlinks + brand |

## 8. Keep improving

- Add more class/board pages over time
- Update FAQ when parents ask new questions
- Keep phone and WhatsApp details consistent everywhere (NAP)
